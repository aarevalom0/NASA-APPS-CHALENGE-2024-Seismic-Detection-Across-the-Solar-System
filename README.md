
## Guía para Usar un Modelo en un Repositorio `.ipynb`
# Puedes abrilo desde un jupyter notebook navega hasta el archivo .ipynb que contiene el modelo y haz clic en él para abrirlo.

Una vez que el notebook esté abierto, puedes ejecutar las celdas de la siguiente manera:

Ejecutar una celda: Selecciona la celda y presiona Shift + Enter.
Ejecutar todas las celdas: Ve al menú y selecciona Cell -> Run All.
